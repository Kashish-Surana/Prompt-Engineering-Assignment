# BrandName Website

A modern, responsive front-end website built using **HTML** and **Tailwind CSS**. This project demonstrates clean, semantic design with a fully responsive layout for desktop, tablet, and mobile.

## Features

- **Navigation Bar**
  - Sticky top
  - Desktop links + mobile hamburger menu
  - Responsive slide-down menu

- **Hero Section**
  - Large headline with subtext
  - Primary CTA button
  - Optional illustration/image placeholder
  - Split layout on desktop, stacked on mobile
